Embedding jsreport

This is the prototype for jsreport end user customization feature. You can go through five scenarios that came to my mind jsreport could support.

Source code for jsreport fork - https://github.com/pofider/jsreport/tree/embedding
Source code for example - https://github.com/pofider/jsreport-embedding
Issue and discussion - https://github.com/jsreport/jsreport/issues/20

  1. Customization with input data
  2. Customization with loading data via script
  3. Client side html widgets
  4. Report script authentication using cookie
  5. Restricting access to the report templates
